# **TETRIS**

## 실행방법
'clond or download'로 파일 저장

`python tetris.py`을 입력하여 실행

## 조작법
- ← : 왼쪽 이동
- → : 오른쪽 이동
- ↑ : 오른쪽으로 회전
- ↓ : 한칸씩 내려감
- 스페이스바 : 바닥으로 내리기
- P 키 : 게임 정지
- Z 키 : 다음 블록으로 바꾸기
